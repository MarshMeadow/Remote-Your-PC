# 🖥️ Remote-Your-PC

A curated list of tools to remotely access, control, or stream your PC from anywhere.

Useful for:
- 💻 Remote desktop control
- 🎮 Game streaming
- 🛠️ IT support & administration
- 📡 Accessing your home PC on the go

---

## 🚀 Popular Remote Desktop Tools

### 🌐 TeamViewer
Cross-platform remote control and support tool.

👉 https://www.teamviewer.com/en-us/download/windows

---

### ⚡ Parsec
Low-latency remote desktop designed for gaming and creative work.

👉 https://parsec.app/downloads

---

### 🎮 Rainway
Game streaming platform that lets you play PC games remotely.

👉 https://rainway.com/gaming/download

---

### 🌙 Moonlight Game Streaming
Open-source game streaming client using NVIDIA GameStream technology.

👉 https://moonlight-stream.org/#

---

### 🧩 MeshCentral
Self-hosted remote management and device control platform.

👉 https://meshcentral.com/info/downloads.html

---

### 🔧 Remotely (Open Source)
Web-based remote desktop and support tool.

⚠️ May require setup and hosting

👉 https://app.remotely.one/downloads  
👉 https://github.com/immense/Remotely

---

## ⚙️ Self-Hosted / Advanced Options

If you want full control over your remote setup:

- 🧱 MeshCentral (enterprise-level self-hosting)
- 🪛 RustDesk (self-host relay server)
- 🌐 Remotely (web-based control panel)
- 🔒 Tailscale + RDP (secure private network access)

---

## 🧠 Notes

- Some tools require installation on both devices
- Gaming tools (Parsec, Moonlight) focus on low latency
- Self-hosted tools require setup but offer more privacy
- Always enable strong passwords + 2FA where available
